## Easter poster & video assets (placeholders)

This folder is reserved for **Easter** card media for the CardRoots homepage rebrand.

You can upload real MP4/JPG (or PNG/WEBP) assets here using the following suggested naming pattern:

| File (video)                          | File (poster)                           | Suggested card key     | Example title         |
|--------------------------------------|-----------------------------------------|------------------------|-----------------------|
`easter-bunny.mp4`                     | `easter-bunny.jpg`                      | `easter-bunny`         | Easter Bunny          |
`easter-garden.mp4`                    | `easter-garden.jpg`                     | `easter-garden`        | Easter Garden         |
`easter-sunrise.mp4`                   | `easter-sunrise.jpg`                    | `easter-sunrise`       | Easter Sunrise        |
`easter-eggs.mp4`                      | `easter-eggs.jpg`                       | `easter-eggs`          | Easter Eggs           |

Notes:

- These files **do not exist yet** – they are placeholders so you know which filenames to upload.
- When wiring the app, use paths like:
  - `/posters/easter/easter-bunny.mp4`
  - `/posters/easter/easter-bunny.jpg`
- It’s safe to add more cards – just follow the same `easter-{name}.mp4` / `easter-{name}.jpg` pattern.
